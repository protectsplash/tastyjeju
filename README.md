# airbnb-clone

## 멤버
Front-end : Splash, YunseCode

## Front-end Goal
* Google Maps를 통하여, 제주도의 맛집들을 소개하는 맛지도를 완성하려 합니다.

## 페이지 명세
### 메인 페이지 (/)
* 제주 공공 데이터를 통해서 제주도 음식점들의 위치를 google Maps에 표시하고, 보여줍니다.
